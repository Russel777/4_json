# Prettify JSON

Python module for load JSON file and prettify it.

# Quickstart

Example of script launch on Linux, Python 3.5:

```bash

$ python pprint_json.py <path to file>
{
	"DayOfWeek": "понедельник",
	"Hours": "09:00-22:00"
}

```
